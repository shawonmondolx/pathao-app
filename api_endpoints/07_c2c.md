# C2C (Customer-to-Customer) Pickup

**Base URL:** `https://api-hermes.pathao.com/talaria`

---

## GET /api/v1/user/c2c-pickup

Get the C2C pickup assignment list.

**Bundle constant:** `C2C_PICKUP_LIST`  
**Auth required:** Yes

### Success Response `200`

```json
{
  "data": [
    {
      "run_route_order_id": 77,
      "sender_name": "Alice",
      "sender_phone": "01800000001",
      "pickup_address": "House 5, Block B, Dhaka",
      "recipient_name": "Bob",
      "amount": 300,
      "status": "pending"
    }
  ]
}
```

---

## GET /api/v1/user/c2c-pickup/:runRouteOrderId

Get details for a specific C2C pickup order.

**Bundle constant:** `C2C_PICKUP_DETAILS`  
**Auth required:** Yes

### URL Parameters

| Param | Type | Description |
|-------|------|-------------|
| `runRouteOrderId` | int | C2C run route order ID |

---

## POST /api/v1/user/c2c-pickup

Confirm/update a C2C pickup attempt.

**Bundle constant:** `C2C_PICKUP_CONFIRM_URL`  
**Auth required:** Yes

### Request Body (JSON)

```json
{
  "run_route_order_id": 77,
  "status": "collected",
  "cash_collected": 300
}
```

---

## PATCH /dispatches/attempts/attemptId/status

Update the status of a C2C dispatch attempt (OMS endpoint).

**Bundle constant:** `UPDATE_C2C_PICKUP_ATTEMPT_URL`  
**Base URL:** `https://api-hermes.pathao.com/api/agents/v1/oms`  
**Auth required:** Yes

> ⚠️ This endpoint uses a **different base URL**:  
> Full path: `https://api-hermes.pathao.com/api/agents/v1/oms/dispatches/attempts/attemptId/status`

### Request Body (JSON)

```json
{
  "status": "collected",
  "attempt_id": "ATTEMPT123"
}
```
